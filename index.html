Flickr Findr Help

Hey there! You can use this web application to search for Flickr images, without going directly to Flickr. Neat, huh?

You can search for images in general by typing keywords (like a title, tag, or description) into the left-hand box.

You can search for images from a certain user by typing their Flickr username into the right-hand box.

You can also do both, to search for certain kinds of images from a specific user (like all photos of "sky" by user "shutterjack").

If you only want to see images uploaded before and/or after some date, enter the date(s) into the smaller text boxes. Dates should be entered in the format yyyy-mm-dd (year-month-day).

To close this box, click the "X" button in the top-right corner.

If you can't find any photos you like, try a different search!


This application was built by Sarah Green for the User Interface Design course at Columbia University in Fall 2014.
